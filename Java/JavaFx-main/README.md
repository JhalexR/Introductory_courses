# 🎨 JavaFX Desktop Application

## 📖 Overview

This repository contains a desktop application built using JavaFX, designed to provide a graphical user interface (GUI) for a fast-paced Morse code learning game. The application features a reference table of the Spanish alphabet and its corresponding Morse code. Players are challenged to translate words into Morse symbols within a 60-second time limit, earning points for every correct word completed.

The project demonstrates:
+ Object-Oriented Programming (OOP)
+ MVC design principles
+ Event-driven UI development
+ JavaFX component integration

## 🚀 How to Run the Project

### Option 1: Using an IDE (Recommended)

1.	Clone the repository:
git clone https://github.com/JhalexR/JavaFx.git

2.	Open the project in:
	+ IntelliJ IDEA
	+ Eclipse
	+ VS Code (with Java Extension Pack)

3.	Make sure JavaFX SDK is configured.

4.	Run the Main class.
________________________________________

### Option 2: Using Maven (if applicable)
mvn clean javafx:run

________________________________________

### Option 3: Using Gradle (if applicable)
gradle run

________________________________________

## ⚙️ JavaFX Configuration (If Not Using Maven/Gradle)
If running manually, make sure to include VM options:
--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml
Replace /path/to/javafx/lib with your JavaFX SDK location.
